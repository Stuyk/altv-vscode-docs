# player.armour

**Description**

Used to get or set a player's armour.

**Syntax**

```ts
player.armour: number;
```

**Example Usage**

```js
const currentArmour = player.armour;
player.armour = 100; // Set to full armour.
player.armour = 0; // Set armour to zero.
```
