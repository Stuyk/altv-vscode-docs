# player.setWeather

**Description**

Used to set a player's weather.

[Weather IDs & Names](../articles/tables/weather.md)

_If you wish to do weather transitions you must do it client-side._

**Example Usage**

```js
// weather id
player.setWeather(0); // Extra Sunny
```
