# player.setWeather

Used to set a player's weather. [Weather Info](https://wiki.altv.mp/wiki/GTA:Weather)

_If you wish to do weather transitions you must do it client-side._

**Example Usage**

```js
// weather id
player.setWeather(0); // Extra Sunny
```
