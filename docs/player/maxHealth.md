# player.maxHealth

**Description**

Used to get or set the player's Maximum Health.
Makes it larger than 199.

**Example Usage**

```js
const maxHealth = player.maxHealth;
console.log(`Their max health is: ${maxHealth}`);
player.maxHealth = 1999;
```
