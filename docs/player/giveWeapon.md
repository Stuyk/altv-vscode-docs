# player.giveWeapon

Used to give a player a weapon based on a hash from this [page](https://wiki.altv.mp/wiki/GTA:Weapon_Models).

**Example Usage**

```js
// Combat shotgun.
// 9999 Ammo
// Equip immediately
player.giveWeapon(0x5a96ba4, 9999, true);
```
