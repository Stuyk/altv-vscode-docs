# player.removeWeapon

Used to remove a player weapon based on a weapon from this [page](https://wiki.altv.mp/wiki/GTA:Weapon_Models).

**Example Usage**

```js
// Combat shotgun.
player.removeWeapon(0x5a96ba4);
```
