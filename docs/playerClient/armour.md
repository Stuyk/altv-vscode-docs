# alt.Player.local.armour

Used to obtain a player's armour.

**Syntax**

```ts
readonly armour: number;
```

**Example Usage**

```js
const currentArmour = alt.Player.local.armour;
```

_alt.Player.local can also be a **player instance** from server_
