# alt.Player.local.id

Get the player's identification number for server-side or client-side.

**Syntax**

```ts
readonly id: number;
```

**Example Usage**

```js
const localPlayerID = alt.Player.local.id;
```

_alt.Player.local can also be a **player instance** from server_
